# Fragment

## What it is
Inlines content from a referenced fragment path into the page.

## Authoring (table shape)
- **Row 1**: block name `Fragment` (optional variants in same cell)
- **Row 2**: link to the fragment (typically a path under `/fragments/`)

## Implementation
- **Code**: `blocks/fragment/fragment.js`
- **Styles**: `blocks/fragment/fragment.css`
