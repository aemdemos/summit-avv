# Cards

## What it is
A grid/list of cards, usually with image + text + link patterns.

## Authoring (table shape)
- **Row 1**: block name `Cards` (optional variants in same cell)
- **Rows 2+**: one card per row (commonly 2–3 columns, depending on the chosen pattern)

## Implementation
- **Code**: `blocks/cards/cards.js`
- **Styles**: `blocks/cards/cards.css`
