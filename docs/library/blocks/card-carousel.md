# Card Carousel

## What it is
A horizontally scrollable/steppable carousel of cards.

## Authoring (table shape)
- **Row 1**: block name `Card Carousel` (optional variants in same cell)
- **Rows 2+**: one card per row (columns depend on the card structure used by this project)

## Implementation
- **Code**: `blocks/card-carousel/card-carousel.js`
- **Styles**: `blocks/card-carousel/card-carousel.css`
