# Embed

## What it is
Embeds third-party content (for example video, maps, iframes) from a URL.

## Authoring (table shape)
- **Row 1**: block name `Embed` (optional variants in same cell)
- **Row 2**: embedded URL (and optional additional settings depending on variant)

## Implementation
- **Code**: `blocks/embed/embed.js`
- **Styles**: `blocks/embed/embed.css`
