# Form

## What it is
A configurable form block for collecting user input.

## Authoring (table shape)
- **Row 1**: block name `Form` (optional variants in same cell)
- **Rows 2+**: form fields / configuration rows (field type, label, name, options, etc.)

## Implementation
- **Code**: `blocks/form/form.js`
- **Styles**: `blocks/form/form.css`
