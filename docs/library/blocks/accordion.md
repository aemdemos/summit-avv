# Accordion

## What it is
An expandable/collapsible list of items (title + content).

## Authoring (table shape)
- **Row 1**: block name `Accordion` (optional variants in same cell)
- **Rows 2+**: 2 columns per item
  - **Column 1**: item title
  - **Column 2**: item content (text/media/links)

## Implementation
- **Code**: `blocks/accordion/accordion.js`
- **Styles**: `blocks/accordion/accordion.css`
