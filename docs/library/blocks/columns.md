# Columns

## What it is
A responsive multi-column layout for grouping content side-by-side.

## Authoring (table shape)
- **Row 1**: block name `Columns` (optional variants in same cell)
- **Rows 2+**: one row per “column set”; each cell becomes a column

## Implementation
- **Code**: `blocks/columns/columns.js`
- **Styles**: `blocks/columns/columns.css`
